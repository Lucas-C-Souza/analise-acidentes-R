# Análise de Acidentes Rodoviários 2024 - Linguagem R

]Este projeto faz parte de uma aula prática de Probabilidade e Estatística. 
O objetivo é analisar dados reais de acidentes para identificar padrões de causa, 
localização e probabilidade.

## 🛠️ Tecnologias Utilizadas
* Linguagem R
* VS Code (Ambiente de Desenvolvimento)
* Bibliotecas: `dplyr` e `ggplot2` 

## 📊 Principais Insights
* O estado com maior número de acidentes foi **MG**].
* A probabilidade de acidentes em condições de **Céu Claro** é de **65,15%**.
* O período do **Pleno dia** concentra a maioria das ocorrências.
